# AXON Protocol — Smart Contracts ⚡

On-chain identity, wallet, and escrow infrastructure for AI agents.

## Contracts

| Contract | Description |
|----------|-------------|
| `AgentRegistry.sol` | ERC-721 based Agent ID — register, verify, track AI agents on-chain |
| `AgentWallet.sol` | Smart wallet with spending limits, whitelists, and emergency freeze |
| `SimpleEscrow.sol` | Trustless escrow for agent-to-agent transactions with dispute resolution |

## Tech Stack

- Solidity ^0.8.20
- OpenZeppelin Contracts
- Target: Base (Ethereum L2)

## Status

🚧 **In Development** — Not audited, not for production use yet.
